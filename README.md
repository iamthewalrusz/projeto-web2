# Ágora – Projeto Web II

---

## Pré-requisitos

- Node.js instalado (versão 18+ recomendada)
- Conta e banco configurados no NeonDB

---

## Configuração do back-end
1. Abra um terminal (**cmd** ou PowerShell).
2. Vá até a pasta do backend: cd backend
4. Instale as dependências: npm install
5. Inicie o servidor back-end: npm run dev
6. O back-end ficará disponível em: http://localhost:4000

## Configuração do front-end
- Abra um terminal: cmd
- Acesse a pasta frontend: cd frontend
- Execute o comando: npm install
- Execute o comando: npm start
- Acesse o servidor em: http://localhost:3000
